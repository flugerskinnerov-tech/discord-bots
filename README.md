# Discord Боты

Три бота для Discord:
- 🏰 **Clans Bot** - система кланов (префикс *)
- 📋 **Logging Bot** - логирование событий (префикс +)
- 🔒 **Private Channels Bot** - приватные каналы (префикс !)

## Установка
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите всех ботов: `python run_all.py`
